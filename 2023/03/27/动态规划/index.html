3. 目标和494

主要：装满这个背包有多少种方法

递归代码：dp[j] += dp[j-nums[i]]

dp[5] = dp[4] + dp[3] + dp[2] +dp[1] +dp[0];

本题要如何使表达式结果为target，分割成两个子集

既然为target，那么就一定有 left组合 - right组合 = target。

left + right = sum，而sum是固定的。right = sum - left

公式来了， left - (sum - left) = target 推导出 left = (target + sum)/2 。

target是固定的，sum是固定的，left就可以求出来。

此时问题就是在集合nums中找出和为left的组合。

子序列问题

1. 最长递增子序列300（一维不连续）

2 .最长连续递增子序列674（一维连续）

一维dp数组，一层遍历，明确是求几种方式还是要求长度（考虑是否需要加一）

需要连续，只需要比较i和i-1进行比较，不需要遍历前面所有

3.最长重复子数组718（二维连续，相等时一起回退）

  1. 明确dp数组的含义
    • 定义二维dp数组，
    • dp[i][j]表示以i-1为结尾的nums1和以j-1为结尾的nums2两个数组的最长子数组的长度
    • 注意：如果这里是以i，j为结尾的话，在初始化的时候要对dp[i][0]和的dp[0][j]作遍历， 有相等值要对其赋值为1，这样就比较麻烦
  2. 确定状态转移方程
    • 当nums1[i-1]和nums2[j-1]相等，则前一个状态应该是i和j一起回退1
    • dp[i][j] = dp[i-1][j-1]+1
  3. 初始化
    • dp[i][0]和dp[0][j]都应该初始化为0
  4. 确定遍历顺序
    • 两层for循环，先遍历nums1或者nums2均可
  5. 打印dp数组

4.最长公共子序列1143(二维不连续，相等时一起回退，不等时各退一取最大值)
