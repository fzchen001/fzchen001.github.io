Gitee上传本地项目

这次示例是向gitee上上传本地项目，Gitee 是开源中国推出的基于 Git 的代码托管服务，本质和github是一样的，不过访问速度比github快得多。

一、新建仓库

gitee中新建仓库跟github步骤一样

复制该地址之后会用到

二、本地上传

1、首先随便在一个位置打开git bash

2、使用命令git clone url 拉取 远程仓库的.git文件

3、拉取成功后打开文件夹

4、将.git文件剪切到上传项目的文件列表中

5、开始上传

  1. 在该项目文件夹中打开git bash
  2. 输入git add . 将项目文件添加到缓存区
  3. 输入git commit -m ‘firsh commit’ 将项目提交到本地仓库
  4. 输入git push 推到远程仓库

在gitee中打开项目
上传成功！
